

/*Nick white
Greedy solution
For every integer check if there is something lesser ahead
if yes pop the current integer from stack

Striver Explanation
IMP - PSE - front to back - monotonic increasing Stack
======================================================
Same as PSE solution
1 difference is we can remove only k digits, so for every stack.pop() do k--, and break the while when k == 0
1. PSE with k tracking
    traverse string using index or num.toCharArray
    while (!stack.isEmpty() && k > 0 && stack.peek() > digit) 
    pop, k--
2.  if k > 0, k still greater than 0, some stack peek values are == digit , stack.peek() == digit
    we need to remove them
3. pop everything from stack, add to stringBuilder, reverse sb
4. Remove leading zeroes from sb, 
    while(sb.length() > 1 && sb.charAt(0) == '0'){
            sb.deleteCharAt(0);
    }
======================================================

Front to back Pushing Increasing order elements to the stack - from top to bottom the stack will be monotonically decreasing
        /
      /
    /
  /
/
If an element is trying to reduce the order, remove the stack peek

*/
class Solution {
    public String removeKdigits(String num, int k) {
        int size = num.length();
        
        //edge case if size of num is k , we have remove all digits
        if(k == size) return "0";
        
        Stack<Character> stack = new Stack<>();
        
        int index = 0;
        while(index < size){
            while(k>0 && !stack.isEmpty() && stack.peek() > num.charAt(index)){
                stack.pop();
                k--;
            }
            
            stack.push(num.charAt(index));
            index++;
        }
        
        /*Ending number is greater, or last numbers are equal and we havent deleted all k
        number of characters*/
        //This is for cases like "112" and we have to delete 1 char, which has to be last
        
        //This is for cases like "121111" or "332333" and we have to delete 3 chars
        //Here 2 will be deleted, but other ones will not be
        //where peek character is not greater than character to be added
        
        while(k > 0){
            stack.pop();
            k--;
        }
        
        StringBuilder sb = new StringBuilder();
        while(!stack.isEmpty()){
            char current_char = stack.pop();
            sb.append(current_char);
        }
        sb.reverse();
        
        //removing leading zeroes //if Sb has any
        while(sb.length() > 1 && sb.charAt(0) == '0'){
            sb.deleteCharAt(0);
        }
        
        return sb.toString();
        
    }
}
