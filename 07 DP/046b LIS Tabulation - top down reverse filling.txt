/*
Extrapolating from recursion solution
if (i == n) return 0;
dp[i][*] = 0

From previous recur + memo solution
Whenever we convert recursion+memo solution to dp 
DO COORDINATE SHIFT OF i as well with prev

Sizes of dp
1. prev starts from -1, to make it 0 based prev+1 size
2. base case i=n, so index should span from 0 to n, i.e. n+1 size
dp[n+1][prev+1]


Top down dp filling - reverse Filling
Steps
1. Create a dp array of size n+1 and n+1
2. Base case, for all prevs at i=n, 0, 0 length of subsequence
2. For in for loop, top down, for i=n-1 to i>=0
    2.1 - exclude case notTake
     exclude - current index - dont add 1
    int exclude = dp[i + 1][prev + 1];
    2.2 - include case - take
    include current index - if curr > prev, make curr as prev
    if (prev == -1 || nums[i] > nums[prev]) {
        include = 1 + dp[i + 1][i + 1];
    }
*/

import java.util.*;

public class LongestIncreasingSubsequence {

    public static int findLIS(int[] nums) {
        int n = nums.length;

        int[][] dp = new int[n + 1][n + 1];

        // 🔹 Base Case: i = n
        for (int prev = -1; prev < n; prev++) {
            dp[n][prev + 1] = 0;
        }

        // 🔹 Fill DP
        for (int i = n - 1; i >= 0; i--) {
            for (int prev = i - 1; prev >= -1; prev--) {

                // exclude - current index - dont add 1
                int exclude = dp[i + 1][prev + 1];

                // include current index - if curr > prev, make curr as prev
                int include = 0;
                if (prev == -1 || nums[i] > nums[prev]) {
                    include = 1 + dp[i + 1][i + 1];
                }

                dp[i][prev + 1] = Math.max(include, exclude);
            }
        }

        return dp[0][-1+1]; // prevIndex = -1 → shifted to 0
    }
}
