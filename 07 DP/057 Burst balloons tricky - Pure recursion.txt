
/*
Striver explanation - TOP DOWN BURSTING - BURST LAST FIRST then RECUR
NOTE IMP -> Explanation

1 b1 b2 b3 b4 b5 1
Consider bursting b3 now [1,b1,b3] [b4 b5 1]
Now b3s right is b4 but its in another array
If we burst first and then split, the left array's right element depends of right arrays left element
So both arrs cannot be independent


Solution is
Reverse the logic, instead of bursting first and splitting into 2 independent arrays
Consider the currentIndex to be bursted at the last
1 b1 b2 b3 b4 b5 1 - arr
  i    ind    j    - indices
so if b3 is burst at last 
Points  = 1 x b3 x 1 (because rest b1,b2,b4,b5 would have burst earlier only)
Points  = arr[i-1] x b3 x arr[j+1] 
Now recur on both left and right independently, 
as if any of them bursts last, its independent of the other splitted array
Now the split is arr1 [1 b1 b2];  b3(burst);  arr2 [b4 b5 1]
So in the next recurfunctions recurFunc(arr1) and recurFunc(arr2), we only need the last burst b3 val on both sides
TOP DOWN BURSTING

NOTE: Any index can be last burst so need a for loop from i -> j in the recurFunction
*/

/*
Steps: 
1. Padded Array/ Expanded Array
  - Built an expanded array with size n + 2. 
  - add 1 at both ends, arr[0] and arr[n+1], 
  - fill the rest of the arr from index=1 to index=n
2. RecurFunction
  2.1 base case if i > j return 0, no points
  2.2 for burstIndex from i to j
      - compute currPoints = curr * left of first * right of last
      currPoints = arr[i-1] * arr[lastBurstIndex] * arr[j-1]
      - compute totalPoints -> currPoints + recurLeft + recurRight
      totalPoints = currPoints + recurFunction(i, lastBurstIndex - 1) + recurFunction(lastBurstIndex + 1, j)
      - maintain a maxPoints var maxPoints = Math.max(maxPoints, totalPoints)
  2.3 return maxPoints
*/

public class BurstBalloons {

    public static int solve(int[] arr, int i, int j) {
        // no balloon in range
        if (i > j) return 0;

        int maxCoins = 0;

        // try each balloon as LAST to burst
        for (int lastBurstIndex = i; lastBurstIndex <= j; lastBurstIndex++) {
        
            int coins = arr[i - 1] * arr[lastBurstIndex] * arr[j + 1];
        
            int total = solve(arr, i, lastBurstIndex - 1)
                      + solve(arr, lastBurstIndex + 1, j)
                      + coins;
        
            maxCoins = Math.max(maxCoins, total);
        }

        return maxCoins;
    }

    public static int maxCoins(int[] nums) {
        int n = nums.length;

        // build padded array
        int[] arr = new int[n + 2];
        arr[0] = 1;
        arr[n + 1] = 1;

        for (int i = 0; i < n; i++) {
            arr[i + 1] = nums[i];
        }

        return solve(arr, 1, n);
    }

    public static void main(String[] args) {
        int[] nums = {3, 1, 5, 8};

        System.out.println(maxCoins(nums));
    }
}
