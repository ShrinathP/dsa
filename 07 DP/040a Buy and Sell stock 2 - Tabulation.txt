Striver Explanation 
Gemini code
Create a dp array with size n = prices.length cross 2 states i.e. 0, 1
Base case - at the end n,0 and n,1 profit is 0
Top down Table filling - opposite of bottom up recursion.
for i-1 to 0, profit will be max at 0th index, 0th state i.e. can buy state

class Solution {
    public int maxProfit(int[] prices) {
        int n = prices.length;
        // dp[i][0] means we cannot buy (we need to sell first)
        // dp[i][1] means we can buy (we currently hold no stock)
        int[][] dp = new int[n + 1][2];

        // Base case: on day n (after the last day), profit is 0
        dp[n][0] = 0;
        dp[n][1] = 0;

        // Iterate backwards from the second-to-last day down to day 0
        for (int i = n - 1; i >= 0; i--) {
            for (int buy = 0; buy <= 1; buy++) {
                int profit = 0;
                if (buy == 1) {
                    // Option 1: Buy the stock, Option 2: Do nothing
                    int buyStock = -prices[i] + dp[i + 1][0];
                    int skip = dp[i + 1][1];
                    profit = Math.max(buyStock, skip);
                } else {
                    // Option 1: Sell the stock, Option 2: Do nothing
                    int sellStock = prices[i] + dp[i + 1][1];
                    int skip = dp[i + 1][0];
                    profit = Math.max(sellStock, skip);
                }
                dp[i][buy] = profit;
            }
        }
        
        // Return max profit starting on day 0 with permission to buy
        return dp[0][1];
    }
}
