
1. Like recursion backtracking playlist, but 
there we do recursion from 0 to n-1, 
here n-1 to 0


Prefer (m-1,n-1) → backward style traversal as it transitions cleanly into DP versions


class Solution {
    public int uniquePaths(int m, int n) {
        return f(m - 1, n - 1);
    }

    int backtrack(int i, int j) {
        if (i == 0 && j == 0) return 1;
        if (i < 0 || j < 0) return 0;
        int up = backtrack(i - 1, j);
        int left = backtrack(i, j - 1);
        return  up + left;
    }
}
