/*
Minimum cost to cut the stick - imp
Striver explanation and video

Problem Statement (brief)
You are given:
A stick of length n
An array cuts[] → positions where you must cut

Rules:
You can cut in any order
Cost of a cut = length of current stick segment
Return minimum total cost

NOTE: Why sorting is imp ?
Consider cuts arr = [3,4,5,2*] and n=7
so length of stick is 7 
i.e 1,2,3,4,5,6,7
i.e 0,1,2,3,4,5,6 indexes
once you cut at 3, we have [1,2,] [3,4,5,6,7],
BUT for the last cut 2* we still have to go to the left subarray
So Sorting the cuts array is important
If you sort it, the next cuts will only be dependent on the right subarray

Consider cuts arr = [3,4,5,2] and n=7
IMP - include the cut portion in both subarrays
1. Sort it arr = [2,3,4,5]
                  i     j
2. Add indexes of the last cuts to left and right of the arr
    arr = [0,2,3,4,5,7]
             i     j
3. Total cost to cut = arr[j+1] - arr[i-1]
4. Recur on both sides = recur(i, ind-1) + recur(ind + 1, j)
5. Next cut 2, arr1[0,2] arr2[2,3,4,5,7] -> notice we include cut number portion in arr1 and arr2
6. min = Math.min(min, cost)
*/

/*
Steps
1. Create new cuts arr with size cutsArr.length + 2, 
    - add 0 and n to both ends
    - copy the existing cutsArr to new cuts
    - sort the new cuts array
2. recurFunction(i, j, cutsArr)
   recurFunction(1, n, cutsArr)
    - Base case -> if i crosses j, no cut possible return 0
        if(i > j) return 0
    - init minCost = Int.MAX_VALUE
    - For every curIndex from i to j
        - calc currCost = curr stick length (before cutting)
                        + recur from i to left of cut (i, cutIndex - 1)
                        + recur from right of cut to j (cutIndex, j)
        - (cuts[j+1] - cuts[i-1])
            + recurFunction(i, cutIndex - 1 , cuts)
            + recurFunction(cutIndex + 1, j, cuts)
        minCost  = Math.min(minCost, cost)
    
*/


import java.util.*;

public class MinCostCutStick {

    public static int solve(int i, int j, int[] cuts) {
        // No cuts possible
        if (i > j) return 0;

        int minCost = Integer.MAX_VALUE;

        // Try every possible cut/partition index k
        for (int cutIndex = i; cutIndex <= j; cutIndex++) {

            int cost = (cuts[j + 1] - cuts[i - 1])  // current stick length
                     + solve(i, cutIndex - 1, cuts)        // left part
                     + solve(cutIndex + 1, j, cuts);       // right part

            minCost = Math.min(minCost, cost);
        }

        return minCost;
    }

    public static int minCost(int n, int[] cutsArr) {
        int c = cutsArr.length;

        // create new cuts array + 2 size, add 0 and n at 2 ends
        int[] cuts = new int[c + 2];
        cuts[0] = 0;
        cuts[c + 1] = n;

        for (int i = 0; i < c; i++) {
            cuts[i + 1] = cutsArr[i];
        }

        Arrays.sort(cuts);

        return solve(1, c, cuts);
    }

    public static void main(String[] args) {
        int n = 7;
        int[] cuts = {1, 3, 4, 5};

        System.out.println(minCost(n, cuts));
    }
}
