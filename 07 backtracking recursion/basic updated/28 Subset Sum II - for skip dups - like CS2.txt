/*

Chatgpt soln
Given int[] nums (may contain duplicates), return all unique subsets.

The input array may contain duplicates.
You need to return all unique subsets (not just subset sums).


=================================================================
Steps
IMP CUE - Only 2 additions to the for loop Subsets Power Set Problem
* Addition 1: Sorting needed to Skip duplicates
Arrays.sort(arr); // required
* Addition 2: Skipping Algorithm
if (i > index && arr[i] == arr[i - 1]) continue; // 🔥 only change
=================================================================


✅ Algorithm Steps:
1. Sort the array:
Helps detect duplicates (e.g., [1,2,2] → sorted remains same).
2. Start backtracking:
At each index:
- You can include the element.
- Or skip to the next.
3. Skip duplicates:
- When looping, if nums[i] == nums[i-1] and i > index, skip nums[i].
- This ensures that from a given recursive level, duplicate numbers don't start new identical subsets.
4. Add current subset to result at every call.

✅ Why i > index is necessary:
It ensures we only skip duplicates at the same recursive level, not across levels.
IMP: Check skip duplicates logic in 28 file number

Question
[1,2,2] wont be there right ? as 2 and 2 are duplicates
Good catch—but here’s the subtle (and very important) correction:
❌ [1,2,2] will NOT be removed
✅ It IS a valid subset because the input itself has two 2s

Key Rule (burn this in)
If array = [1,2,2]
Then [2,2] and [1,2,2] are valid ✅
But generating [2] twice is invalid ❌
NOTE IMP -> only the starts we are skipping, example for 1,2,2 -> if for 1st 2 we ran, we dont run for 2nd 2
*/


import java.util.*;

class Solution {
    public List<List<Integer>> subsetsWithDup(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        Arrays.sort(nums); // Needed to skip duplicates
        backtrack(nums, 0, new ArrayList<>(), result);
        return result;
    }

    private void backtrack(int[] nums, int index, List<Integer> temp, List<List<Integer>> result) {
        result.add(new ArrayList<>(temp));

        for (int i = index; i < nums.length; i++) {
            // Skip duplicates at the same recursive level
            if (i > index && nums[i] == nums[i - 1]) continue;
            //** only the starts we are skipping, example for 1,2,2 -> if for 1st 2 we ran, we dont run for 2nd 2

            temp.add(nums[i]);
            backtrack(nums, i + 1, temp, result);
            temp.remove(temp.size() - 1); // backtrack
        }
    }
}
