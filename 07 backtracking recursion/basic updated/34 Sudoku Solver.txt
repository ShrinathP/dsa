
/*

Fill incomplete sudoku puzzle

Constraints - for placeable function
1. The digit 1-9 should exactly appear only once in a given row
2. The digit 1-9 should exactly appear only once in a given column
3. The digit 1-9 should exactly appear only once in a given 3x3 board


Striver Explanation

1. loop for each row, for each column
2. check if the content is dot
	2.1 If its dot 
	2.2 Run loop for char 1 to 9 
		2.3 Check for each digit if the number is placeable
	2.4 if it is placeable, place the number at board[i][j]
	2.5 solve the board for other places
	2.6 if solve is true, return true
	2.7 else remove the placed number (backtrack step)

Placeable function
isValid Function
For every row, column number check if 
the number can be placed 
	- along the row, 
	- along the column, 
	- along the 3*3 box

how is row and col calculated in sudoku solver
board[3 * (row / 3) + i / 3][3 * (col / 3) + i % 3] == c function 
Subgrid Origin (3 * (row / 3) and 3 * (col / 3)):
If row is 4, 4 / 3 = 1
Offset within Subgrid (i / 3 and i % 3):
If row is 4, 4 / 3 = 1
Divide by 3 Modulo 3



1. if board == null or board.length == 0, return as it is
2. recurFunction solve
    2.1 for in for, for in in rows, for j in columns
        3.1 if board is blank -> if board[i][j] == '.' - we will try placing from 1 to 9
            - For char = 1 to 9 run loop
                - if(isValid(board, row, col, currChar))
                    - fill board
                    - call recurFunction solve
                    - if recurFunction solve returns true, return true
                 - else backtrack by setting board[i][j] == '.'
            - return false - tried everything from 1 to 9 

IMP - ❗ The key doubt
Why is return false after the for loop instead of inside backtracking?
👉 "I tried ALL digits (1–9), and NONE worked for this cell"

class Solution {
    //Take u forward
    //As we need to find only 1 True solution, 
    //directly return true if found one
    public void solveSudoku(char[][] board) {
        if(board == null || board.length == 0)
            return;
        solve(board);
    }
    
    private boolean solve(char[][] board){
        for(int i=0; i< board.length; i++) {
            for(int j=0; j< board[0].length; j++) {
                //check if place is blank
                if(board[i][j] == '.') {
                    for(char c = '1'; c<= '9'; c++) {
                    //trail 1 through 9
                        if(isValid(board, i, j, c)){
                            board[i][j] = c;
                            
                            //solve for next blank place
                            if(solve(board)) {
                                return true; 
                            } else {
                            //if its solution return true
                            //backtrack
                                board[i][j] = '.'; 
                            //or put back dot , to check for other paths
                            }
                        }
                    }
                    //if nothing returned from for
                    return false;
                }
            }
        }
        return true;
    }
    
    
    //Checking row , column and 3*3 matrix
       private boolean isValid(char[][] board, int row, int col, char c){
        for(int i = 0; i < 9; i++) {
            if(board[i][col] != '.' && board[i][col] == c) return false; //check row
            if(board[row][i] != '.' && board[row][i] == c) return false; //check column
            if(board[3 * (row / 3) + i / 3][ 3 * (col / 3) + i % 3] != '.' && 
board[3 * (row / 3) + i / 3][3 * (col / 3) + i % 3] == c) return false; //check 3*3 block
        }
        return true;
    }
}
